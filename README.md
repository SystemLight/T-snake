# T-snake

Basic development template of Python project

## 用法

1. T-snake默认使用 `pipenv` 作为包管理器，如果没有安装请执行下面命令

```
pip install pipenv

安装后添加环境变量
PIPENV_VENV_IN_PROJECT = 1
```

2. 还原依赖环境

```
pipenv install
```

3. T-snake默认使用阿里源仓库作为安装源
